# SOLANA TOKENS BOT

## Usage

to set up project, you have to first create a `.env` file in `env` directory and add your telegram bot token like in `.env.example` file

## Installation 

### Prerequisites

- Install Docker and docker-compose
- Install Makefile

### Build Docker Image
```bash
make build
```

### Run docker container
```bash
make run
```

### Remove container
```bash
make kill
```
